Musique et Technologie

La musique a toujours avancé avec la technologie. Chaque époque a ses révolutions : l’acoustique et ses instruments, l’électrique qui amplifie et déforme, le numérique qui copie et transforme à l’infini.

Musique et IA

“Après l’électricité et le numérique, l’IA s’impose comme une nouvelle révolution musicale. La vraie question n’est pas comment les machines composent, mais comment nous allons les utiliser.”

Séparation de sources (Stem Separation)

Isoler voix, batterie, basse ou instruments d’un morceau complet grâce au deep learning (DJing ++).

timeline 2020 : Spleeter (Deezer) 2021 : Demucs (Meta) 2023 : Sound Demixing Challenge

Contributions scientifiques majeures

  • Spleeter (Deezer / 2020) (site) (pdf) (github)
  • Demucs (Meta / 2021) (site + exemples) (pdf) (github)
  • Challenge pour chercheurs : Sound Demixing Challenge.

Outils pour musiciens

  • Izotope RX11 (site)
  • Native Instruments Maschine 3 (site)
  • Native Instruments Traktor Pro 4 (site)

Composition automatique

Générer des mélodies, harmonies ou arrangements à partir de règles, d’un style choisi ou d’une description textuelle (Text-to-MIDI).

timeline 1989 : CONCERT (premiers systèmes à règles) 2016 : PerformanceRNN (Magenta) 2019 : MuseNet (OpenAI) 2019 : Music Transformer (Magenta) 2023 : MuseCoco (Tencent)

Contributions scientifiques majeures

  • CONCERT (1989) : génération automatique de mélodies à partir de contraintes harmoniques.
  • Magenta / PerformanceRNN (Google, 2016–2017) (site)
  • MuseNet (OpenAI / 2019) (site + exemples)
  • Music Transformer (Google Magenta / 2019) (site) (pdf)
  • MuseCoco (Tencent AI Lab / 2023) (pdf) (github)

Outils pour musiciens

  • AIVA (2016–) (lien)
  • Amper Music (2017–) (lien)
  • MuseNet (OpenAI / 2019) (lien)
  • Google Magenta Studio (2019–) (lien)

Clonage Vocal (Voice Cloning)

Reproduire la voix d’un chanteur ou d’un comédien grâce à l’IA. Les applications vont du doublage multilingue aux chansons “posthumes”, en passant par la création de voix artificielles réalistes.

timeline 2016 : WaveNet (DeepMind) 2018 : SV2TTS 2020 : Descript Overdub 2023 : VALL-E (Microsoft) 2023 : ElevenLabs 2024 : ACE Studio

Contributions scientifiques majeures

  • WaveNet (DeepMind / 2016) (site) (pdf)
  • SV2TTS (2018) : encoder–synthesizer–vocoder permettant de cloner une voix à partir de quelques secondes d’audio. (pdf) (github)
  • VALL-E (Microsoft / 2023) : synthèse vocale zéro-shot à partir de 3 secondes d’échantillon. (exemples) (pdf)

Outils pour musiciens et créateurs

  • Descript Overdub (2020–) (site)
  • Respeecher (2020–) (site)
  • iSpeech / Replica Studios (2021–) (site)
  • ElevenLabs (2023–) (site)
  • Grimes / Elf.Tech (2023–) (site)
  • ACE Studio (2024–) (site)

Synthèse sonore

Créer de nouveaux sons ou instruments directement à partir de modèles génératifs (au-delà du simple sampling), contrôlables en temps réel.

timeline 2017 : NSynth (Google) 2020 : DDSP (Google) 2021 : RAVE (IRCAM) 2022 : EnCodec (Meta) 2022 : AudioLM (Google)

Contributions scientifiques majeures

  • NSynth (Google Magenta / 2017) (site) (pdf)
  • DDSP (Google / 2020) (site) (pdf) (github)
  • RAVE (IRCAM / 2021) (pdf) (github)
  • EnCodec (Meta / 2022) (pdf) (github)
  • AudioLM (Google / 2022) (pdf) (exemples)

Outils pour musiciens

  • DDSP-VST (Google Magenta) (lien)
  • RAVE-VST (IRCAM / 2022) (lien)
  • Flow Machines (Sony CSL) (lien)

Génération Texte → Audio (Text-to-Audio / Music LM)

Générer automatiquement des sons ou de la musique à partir d’une simple description textuelle (“un piano jazz doux avec un rythme de batterie bossa nova”). Les progrès récents transforment la composition et la production musicale.

timeline 2020 : Jukebox (OpenAI) 2023 : MusicLM (Google) 2023 : MusicGen (Meta) 2024 : Stable Audio Open (Stability AI) 2025 : ACE-Step 2025 : ElevenLabs Music

Contributions scientifiques majeures

  • Challenge pour chercheurs : Benchmark MusicCaps (Google).
  • Jukebox (OpenAI / 2020) (site + exemples) (pdf) (github)
  • MusicLM (Google / 2023) (exemples) (pdf)
  • MusicGen (Meta / 2023) (site + exemples) (pdf) (github)
  • ACE-Step (2025) (github)
  • Stable Audio Open (Stability AI / 2024) (site) (HuggingFace)

Outils pour musiciens

  • Mubert Render (2022–) : génération de musique illimitée via prompt texte, API pour créateurs et marques. (site)
  • Riffusion / Producer.ai (2022–) : génération musicale à partir de spectrogrammes (diffusion), devenue Producer.ai en 2025. (site)
  • Suno (2023–) : génération de chansons complètes à partir d’un prompt texte (voix + accompagnement). (site)
  • Stable Audio (Stability AI / 2023–) : plateforme payante orientée musique de production et sound design. (site)
  • Udio (2024–) : création de morceaux chantés et instrumentaux, avec options de remix et d’édition. (site)
  • ElevenLabs Music (2025–) : service de génération musicale “studio-grade” à partir de texte, usage commercial. (site)
